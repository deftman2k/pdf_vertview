# PDF Vertical Tabs Viewer Release Notes

## 1.0.7
- 최신순 정렬 상태에서 탭 드래그에 반응해 순서가 바뀌던 문제를 차단해 의도치 않은 재정렬과 깜빡임을 막았습니다.
- 최근에 본 문서만 즉시 맨 위로 올리는 방식으로 최신순 정렬을 최적화하여 탭이 많아도 빠르게 갱신됩니다.
- 단일 인스턴스 IPC 흐름을 간소화해 보조 프로세스의 열기 요청을 더 안정적으로 처리합니다.

## 1.0.6
- 세로 탭 컨텍스트 메뉴에 파일명 복사 바로 아래 전체경로 복사 옵션을 추가해 탭의 폴더 경로만 손쉬이 복사할 수 있습니다.
- '다른 이름으로 저장' 대화상자가 마지막 저장 경로를 기억해 도달인 위치를 기본값으로 띄어 줍니다.
- 세로 탭 정렬 옵션을 설정 창으로 옮겨 파일명 오름/내림·최신 모드를 고를 수 있습니다.

## 1.0.5
- 세로 탭 우클릭 메뉴 항목을 `파일명 복사`, `다른 이름으로 저장…`, `파일 위치 열기`, `닫기`로 정리하고 파일명을 곧바로 클립보드에 복사하도록 했습니다.
- PDF 뷰어 영역 우클릭 메뉴에서 저장 관련 항목 문구를 `저장하기`, `다른 이름으로 저장하기`로 다듬었습니다.

## 1.0.4
- UNC 경로에서 문서를 열거나 저장할 때 인증 오류가 발생하던 문제를 해결했습니다.
- UNC 경로가 퓨니코드(xn--) 형식으로 안내되던 메시지를 실제 경로 문자열로 표시하도록 개선했습니다.
- PyOxidizer로 패키징할 때 `RELEASE_NOTES.md` 파일이 누락되던 문제를 수정했습니다.
- PDF 문서를 다른이름 으로 저장 할때, 기존 문서를 삭제하는 설정을 추가했습니다.
## 1.0.3
- 정보 메뉴에 릴리스 정보 항목을 추가하고 애플리케이션 버전을 1.0.3으로 갱신했습니다.
- 창 크기를 조절할 때 탭 패널이 과도하게 넓어지지 않도록 제한을 추가했습니다.
- 암호화된 PDF를 처음 열 때 올바른 암호가 적용되도록 인증 절차를 보완했습니다.
- 탐색기에서 여러 PDF를 드래그했을 때 중복으로 열리던 문제를 수정했습니다.

## 1.0.2
